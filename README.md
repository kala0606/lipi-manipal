# Lipi — the wall is a letter

A proposal for the lobby of the library at MAHE Manipal: a long wall (or a series of framed panels) covered in an invented cipher script — thirty-six letterforms (A–Z, 0–9) generated by a small type system, decodable only via a glossary plate nearby.

**[Read the pitch →](https://kala0606.github.io/lipi-manipal/)**

## In this repo

- `index.html` — the pitch document (self-contained, renders the script live)
- `lipi-wall.html` — the working console: type a message, design a script, export plotter-ready SVGs
- `mauna-lipi.html` — an earlier exploration (a mic-gated "writes only in silence" version) — superseded, kept for reference
- `saxi-plan.js` — exact AxiDraw plot-time calculator, ported from [saxi](https://github.com/nornagon/saxi)
- `mocks/` — architectural mockups (curved wall, framed panel series, decoder detail, glossary plate) and plot-ready SVG exports of the script at wall, panel and glossary scale

## The engine

Every letter is assembled like type, not drawn as a random mark: a spine (stem, arc, oval, arch…) plus a small kit of attachments (bowls, arms, crossbars, dots), combined uniquely per letter under one shared style genome (width, roundness, slant, terminal curl). The whole alphabet regenerates exactly from a single seed number.
